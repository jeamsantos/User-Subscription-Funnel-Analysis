# 📊 User Subscription Funnel Analysis

## 📌 Overview
This project analyzes user behavior in a subscription-based product, focusing on conversion rates, churn, and revenue metrics.

The goal is to simulate how data analysts support product and growth teams by identifying key insights in user journeys.

---

## 🛠️ Tools Used
- SQL (data extraction and aggregation)
- Python (Pandas for data analysis)

---

## 📊 Key Analysis

### Funnel Metrics
- Total Users
- Subscribers
- Conversion Rate
- Churn Rate

### Revenue Analysis
- Total Revenue
- Average Revenue per User (ARPU)

### Segmentation
- Conversion rate by plan (Free, Basic, Premium)

---

## 🔍 Key Insights

- Identified a significant drop-off between total users and subscribers, indicating potential friction in the onboarding process  
- Churn rate highlights opportunities to improve retention strategies  
- Premium users generate higher revenue but may require better retention efforts  
- Conversion rates vary by plan, suggesting different user behaviors and engagement levels  

---

## 🚀 Conclusion

This project demonstrates how data analysis can support decision-making in product and growth teams by:

- Understanding user behavior
- Identifying bottlenecks in conversion funnels
- Supporting revenue optimization strategies
